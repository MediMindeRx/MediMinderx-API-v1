Welcome to MediMindeRx-API!

MediMindeRx is a medication app allowing users to schedule reminders based off location or time.

This is the backend application supplying endpoints for MediMindeRx. To access our endpoints, please refer to the following docs.

Post A New User

Method: POST

https://mediminderx-api.herokuapp.com/api/v1/users

Body: {"name": "John"}

Response: { { "data": { "type": "users", "id": 1, "attributes": { "name": "John" } } }

Get A User

Method: GET

https://mediminderx-api.herokuapp.com/api/v1/users/:id

No Body

Response: { { "data": [ { "type": "users", "id": 1, "attributes": { "name": "John" } } ] }

Get All Users

Method: GET

https://mediminderx-api.herokuapp.com/api/v1/users

No Body

Response: { "data": [ { "type": "users", "id": 1, "attributes": { "name": "John" } }, { "type": "users", "id": 2, "attributes": { "name": "Jane" } } ] }

Edit A User

Method: PUT

https://mediminderx-api.herokuapp.com/api/v1/users

Body: { "name": "Jake", "id": "1" }

Response: { "data": { "type": "users", "attributes": { "name": "Jake" }, "id": 1 } }

Delete A User

Method: DELETE

https://mediminderx-api.herokuapp.com/api/v1/users

Body: { "id": "1" }

Response: { "message": "User has been successfully deleted" }
