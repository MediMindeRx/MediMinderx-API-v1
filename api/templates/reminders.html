Welcome to MediMindeRx-API!

MediMindeRx is a medication app allowing users to schedule reminders based off location or time.

This is the backend application supplying endpoints for MediMindeRx. To access our endpoints, please refer to the following docs.

Choose a datatype

Post A New Reminder

Method: POST

URL: https://mediminderx-api.herokuapp.com/api/v1/reminders

Body: { "user_id": "1", "title": "Soccer Practice", "supplies": "inhaler", "show_supplies": "false" }
